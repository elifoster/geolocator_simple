# simple_geolocator
A simple Geolocating Ruby Gem utilizing IP-API.com.

## Installation
### RubyGems
```
gem install simple_geolocator
```

### Bundler
Add this line to application's Gemfile:

``` ruby
gem('simple_geolocator')
```

And then execute:
```
$ bundle
```
